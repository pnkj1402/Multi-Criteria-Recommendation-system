# Multi-Criteria-Recommendation-system
Basic Deep Matrix Factorization and SVD, With Lasso and Linear regression


Multiple Python files will be implemented in SVD and Basic Deep Matrix Factorization.
..
Each of these techniques is implemented in single and multi-criteria both.
...
Dataset:  Yahoo! Movie data set and TripAdvisor.
...
Lasso and linear regression are used to aggregate the predicted multi-criteria rating to the overall rating...
..
Performance tested on RMSE, MAE and MSE.
